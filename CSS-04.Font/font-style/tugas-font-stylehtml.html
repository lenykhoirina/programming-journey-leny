Contoh Penggunaan Font Style

Paragraf pertama menggunakan ukuran pixel, font Arial, dengan ketebalan normal dan gaya normal.

Paragraf kedua menggunakan satuan em, dengan font Times New Roman dan teks bold.

Paragraf ketiga menggunakan satuan rem, dengan font Courier New dan gaya italic.

Paragraf keempat menggunakan satuan persen, dengan font Verdana dan ketebalan lebih ringan.

Paragraf kelima menggunakan kombinasi berbeda, dengan font Georgia dan gaya oblique.
